## Этап 7 — Обязательные фичи запуска [✅]

> То, что пользователи и рестораны **почти всегда ожидают** от платформы доставки.

| # | Задача | Описание | Статус |
|---|--------|----------|--------|
| **7.1** | **Поиск и фильтры в меню** | По названию, типу (роллы, сеты, вегетарианские), цене, аллергенам/ингредиентам. Критично при большом меню. | ✅ |
| **7.2** | **Гибкая авторизация клиентов** | (а) Вход по телефону + SMS-код. (б) Соцлогины (Google/Apple). (в) Гостевой заказ без регистрации. | ✅ |
| **7.3** | **Несколько способов оплаты** | Наличные (уже есть опция, но нуждается в проверке E2E), Apple Pay / Google Pay через LiqPay. | ✅ |
| **7.4** | **Расширенные уведомления по заказу** | E-mail / SMS / Telegram клиенту: принят → готовится → в доставке → доставлен. Связь с state-machine заказов. | ✅ |
| **7.5** | **Управление запасами (Out of stock)** | Флаг «нет в наличии» для блюд. Быстрое скрытие/показ позиций в меню. Batch-операции. | ✅ |
| **7.6** | **Настройки приёма заказов** | В Brand/Branch: авто-подтверждение vs ручное, окно приёма по времени, минимальный чек доставки. | ✅ |
| **7.7** | **Канал поддержки / обратной связи** | Форма «Проблема с заказом» + общий фидбек. Сбор сигналов, удержание недовольных клиентов. | ✅ |

## Этап 8 — Фичи второй волны (отстройка от конкурентов) [⏳]

> Не критичны на запуске, но дают «вау»-эффект и дополнительную ценность.

| # | Задача | Описание | Статус |
|---|--------|----------|--------|
| **8.1** | **Персонализация рекомендаций** | Блок «Популярное у вас» / «Вы заказывали ранее». Сохранённые предпочтения (острота, без лука и т.п.). | ⏳ |
| **8.2** | **Планирование / предзаказ** | Выбор времени доставки/самовывоза: на сегодня, на вечер, завтра. | ⏳ |
| **8.3** | **Кампании и win-back автоматизация** | Триггерные рассылки (email/Telegram): «давно не заказывали», «у вас N бонусов — успейте потратить». | ⏳ |
| **8.4** | **Продвинутая аналитика для брендов** | Сегментация клиентов (новые/повторные, по чеку), анализ по часам/дням недели, воронка чекаута (корзина → оформление). | ⏳ |
| **8.5** | **Мобильная админка (hamburger-сайдбар)** | Адаптивный сайдбар и таблицы для планшетов/телефонов. Сейчас админка только для десктопа. | ⏳ |
| **8.6** | **Dark Mode** | Тёмная тема в витрине и админке (next-themes). | ⏳ |
